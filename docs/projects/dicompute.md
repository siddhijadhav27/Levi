# DICOMPUTE

## Project Overview

**DICOMPUTE** is a decentralized cloud computing network and supercloud platform built for AI's next frontier. It provides a marketplace where users can tap into global GPU and compute resources at a fraction of the cost of traditional hyperscalers, while providers can monetize their idle hardware.

The DICOMPUTE ecosystem includes a console for deployment, a provider console for onboarding compute resources, a native utility token (DCT), and a comprehensive website and documentation platform. It is positioned as an open-source alternative to centralized cloud providers like AWS, GCP, and Azure — with a focus on AI/ML workloads, Web3 applications, and cost-efficient compute.

## Problem It Solves

Centralized cloud computing suffers from several structural inefficiencies:

- **High Costs** — Hyperscalers set prices to maximize margin, not utilization
- **Vendor Lock-In** — Proprietary APIs and services make migration difficult
- **Limited GPU Access** — AI/ML developers face shortages and high prices for GPU instances
- **Underutilized Hardware** — Millions of GPUs and servers sit idle globally
- **Lack of Transparency** — Pricing is opaque and controlled by a small number of corporations

DICOMPUTE solves this through a **reverse auction marketplace** where:

- **Tenants** deploy applications as Docker containers on globally distributed compute
- **Providers** earn by offering their hardware to the network
- **Prices** are set by open-market competition, maximizing utilization and minimizing cost
- **Settlement** occurs via the DCT token or fiat (credit card) through the DICOMPUTE Console

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | Astro, React, TypeScript, Tailwind CSS |
| **Console** | Web-based deployment UI (DICOMPUTE Console) |
| **Blockchain** | Cosmos SDK-based Proof-of-Stake chain |
| **Token** | DCT (DICOMPUTE Network Token) |
| **Deployment** | Docker containers via SDL (Stack Definition Language) |
| **Payments** | DCT token, credit card (fiat on-ramp), ACT (USD-pegged credits) |
| **Infrastructure** | Kubernetes clusters managed by providers |

## Key Features

- ⚡ **Instant Deployment** — Deploy containers globally in seconds via DICOMPUTE Console
- 💰 **Cost Efficiency** — Up to 80% savings vs. hyperscalers through reverse auction pricing
- 🖥️ **GPU Marketplace** — Access NVIDIA A100s, H100s, and consumer GPUs worldwide
- 🔗 **Web3 Native** — Permissionless, censorship-resistant, community-governed
- 🪙 **DCT Token** — Native utility token for staking, governance, and value exchange
- 🏦 **Fiat On-Ramp** — Credit card payments via DICOMPUTE Console (no crypto required)
- 📦 **1-Click Templates** — Pre-configured deployments for popular AI/ML models
- 🌐 **Global Network** — Compute providers across multiple continents
- 🛠️ **Developer Tools** — CLI, Terraform provider, REST API
- 📊 **Provider Console** — Streamlined UI for becoming a compute provider

## Current Status

**Status:** Live and Growing

DICOMPUTE Network is operational with an active ecosystem of providers and tenants. Key ecosystem projects include:

- **DICOMPUTE Console** — Deployment and management UI
- **DICOMPUTE Provider Console** — Onboarding UI for compute providers
- **DICOMPUTE Chat** — Benchmark conversational LLMs side-by-side
- **DICOMPUTEML** — High-performance AI inference service
- **DICOMPUTEGen** — Text-to-image generation on NVIDIA A100s

The DCT token is listed on major exchanges including Coinbase, Kraken, Upbit, and Crypto.com. The network continues to expand its provider base and GPU availability.
